# 🎉 Convite de Noivado - Anderson & Janine

## Como Publicar no GitHub

### Passo 1: Criar Repositório no GitHub
1. Acesse https://github.com/new
2. Nome do repositório: `convite-noivado-aj`
3. Descrição: "Convite Virtual de Noivado - Anderson & Janine"
4. Selecione: **Public** (para compartilhar o link)
5. Clique em "Create repository"

### Passo 2: Fazer Push do Código

Copie e execute estes comandos no terminal:

```bash
cd /home/ubuntu/convite-noivado-aj

# Adicionar remote (substitua SEU_USUARIO pelo seu usuário do GitHub)
git remote add origin https://github.com/SEU_USUARIO/convite-noivado-aj.git

# Fazer push
git branch -M main
git push -u origin main
```

### Passo 3: Ativar GitHub Pages
1. Vá para: https://github.com/SEU_USUARIO/convite-noivado-aj/settings/pages
2. Em "Source", selecione: **main** branch
3. Clique em "Save"
4. Aguarde alguns segundos
5. Seu convite estará em: https://SEU_USUARIO.github.io/convite-noivado-aj

### Passo 4: Compartilhar o Link
Envie o link para seus convidados via WhatsApp, Email ou Redes Sociais!

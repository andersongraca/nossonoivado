# 💍 Convite de Noivado - Anderson & Janine

Um convite virtual elegante, responsivo e moderno para o noivado de Anderson e Janine, desenvolvido com tecnologias web avançadas e design sofisticado.

## ✨ Características Principais

### Design & UX
- **Responsivo 100%**: Otimizado para celulares, tablets e desktops
- **Mobile-First**: Desenvolvido com foco em dispositivos móveis
- **Paleta de Cores Harmônica**: Verde Sálvia (#B2AC88), Verde Oliva (#708238) e Champagne (#F7E7CE)
- **Psicologia das Cores**: Cores escolhidas para transmitir elegância, natureza e celebração
- **Tipografia Premium**: Fontes do Google Fonts (Playfair Display, Cormorant Garamond, Montserrat)

### Funcionalidades
- 🎵 **Música de Fundo**: Trilha sonora elegante e sofisticada (120 segundos)
- ✨ **Efeitos Visuais**: Animações suaves, partículas flutuantes e transições elegantes
- 📍 **Localização Integrada**: Google Maps embarcado com botões para traçar rota
- 💬 **Confirmação via WhatsApp**: Botão direto para confirmar presença
- 🎨 **Decorações Animadas**: Elementos geométricos com rotação e flutuação
- 📱 **Controle de Áudio**: Botão flutuante para ativar/desativar música
- ♿ **Acessibilidade**: Suporte a preferências de movimento reduzido

### Animações & Interatividade
- Fade-in suave ao carregar
- Scroll reveal animations
- Hover effects nos cards
- Ripple effect nos botões
- Parallax suave
- Partículas flutuantes em background
- Transições fluidas

## 🚀 Como Usar

### 1. Clonar o Repositório
```bash
git clone https://github.com/seu-usuario/convite-noivado-aj.git
cd convite-noivado-aj
```

### 2. Abrir Localmente
Simplesmente abra o arquivo `index.html` em seu navegador:
```bash
# No macOS
open index.html

# No Linux
xdg-open index.html

# No Windows
start index.html
```

### 3. Publicar no GitHub Pages

#### Opção A: Usando GitHub Pages (Recomendado)
1. Faça push do repositório para GitHub
2. Vá para **Settings** do repositório
3. Navegue até **Pages**
4. Em "Source", selecione a branch `main` e pasta `/root`
5. Clique em **Save**
6. Seu convite estará disponível em: `https://seu-usuario.github.io/convite-noivado-aj`

#### Opção B: Usando Vercel (Alternativa)
1. Acesse [vercel.com](https://vercel.com)
2. Clique em "New Project"
3. Importe o repositório do GitHub
4. Clique em "Deploy"
5. Seu convite estará online em minutos

#### Opção C: Usando Netlify (Alternativa)
1. Acesse [netlify.com](https://netlify.com)
2. Clique em "New site from Git"
3. Conecte seu GitHub
4. Selecione o repositório
5. Clique em "Deploy"

### 4. Compartilhar o Link
Após publicar, compartilhe o link com seus convidados via:
- WhatsApp
- Email
- Redes Sociais
- SMS

## 🎨 Personalização

### Alterar Cores
Edite as variáveis CSS no arquivo `styles.css`:
```css
:root {
    --verde-salvia: #B2AC88;
    --verde-oliva: #708238;
    --champagne: #F7E7CE;
}
```

### Alterar Texto
Edite o arquivo `index.html` e procure pelos textos que deseja modificar.

### Alterar Música
Substitua o arquivo `background-music.wav` por outro arquivo de áudio (WAV, MP3, etc).

### Alterar Localização do Mapa
Edite a URL do iframe do Google Maps no arquivo `index.html`:
```html
<iframe src="https://www.google.com/maps/embed?pb=..."></iframe>
```

### Alterar Número do WhatsApp
Edite o link do WhatsApp no arquivo `index.html`:
```html
<a href="https://wa.me/5579999999999?text=...">
```
